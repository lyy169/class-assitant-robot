# Local Processor Skeleton

这个目录用于承接本地电脑端（Processor）正式代码归位。

当前主用代码仍在仓库根目录，后续将按低风险方式逐步迁移到这里。

当前归位策略：

- 先把主链路代码复制到本目录下的结构化位置
- 根目录保留兼容入口，避免已有命令失效
- 等后续稳定后，再逐步收缩根目录入口文件

建议子目录职责：

- `api/`：本地 HTTP 接收接口
- `core/`：YOLO 推理与统计主逻辑
- `train/`：当前训练脚本归位目标
- `training/`：历史占位目录，后续可与 `train/` 合并
- `tools/`：数据构建与辅助工具
- `scripts/`：本地调试与联调脚本
- `config/`：本地端配置文件
- `runtime/`：运行时输出目录映射说明
